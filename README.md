# 🚀 Gerenciador de Tarefas API (V2) - SQLModel & FastAPI

API robusta para gerenciamento de tarefas, desenvolvida para demonstrar competências modernas em Backend com Python. Este projeto utiliza **SQLModel** para uma integração perfeita entre modelos de dados e o banco de dados SQLite.

## 🛠️ Tecnologias
- **Python 3.13+**
- **FastAPI:** Framework de alta performance.
- **SQLModel:** O futuro da integração SQL com Pydantic.
- **SQLite:** Persistência de dados local.

## 🌟 Funcionalidades
- ✅ **CRUD Completo:** Criar, Listar, Atualizar e Deletar tarefas.
- 🔍 **Filtros Dinâmicos:** Busca por status (concluída) e termos de texto.
- 📑 **Documentação Automática:** Swagger UI integrada para testes rápidos.

## 🚀 Como Executar
1. Ative o ambiente virtual: `.\.venv\Scripts\activate`
2. Instale as dependências: `pip install -r requirements.txt`
3. Inicie o servidor: `uvicorn main:app --reload`
4. Teste em: `http://127.0.0.1:8000/docs`
